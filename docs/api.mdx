# Core API

Our API makes working with the Rarimo protocol and smart contract interfaces simply delightful.

:::note

You can work with our API with NO authentication. Its free to use, today and forever.

:::

## Features

- Retrieve a list of compatability blockchains & tokens
- Understand the interface of the contracts
- Build transactions
- Test mode (testnet vs mainnets)

## Access

- BaseURL - https://api.rarify.tokend.io/v1/ (for now)
- Note about design (json, rest, whatever)
