# @rarimo/react-provider

The [@rarimo/react-provider](https://www.npmjs.com/package/@rarimo/react-provider) package provides access to in-browser wallets such as Metamask in React applications.

This package supports the same wallet types as the [@rarimo/provider](./provider) package.

For information about connecting to wallets, see [Connecting to wallets](../overview/connecting-to-wallets).
