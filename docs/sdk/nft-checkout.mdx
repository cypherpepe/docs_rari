# @rarimo/nft-checkout

The [@rarimo/nft-checkout](https://www.npmjs.com/package/@rarimo/nft-checkout) package provides tools that create cross-chain transactions based on the Rarimo protocol.

For information about creating transactions, see [Creating transactions](../overview/creating-transactions).

For example applications, see [rarimo/js-sdk-examples](https://github.com/rarimo/js-sdk-examples/) on GitHub.
