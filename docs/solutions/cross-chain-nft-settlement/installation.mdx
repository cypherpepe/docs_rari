# Installation

The Cross-Chain NFT Settlement component is available as an [npm package](https://www.npmjs.com/package/@rarimo/rarimo-settlement-component), and any new features are released using semantic versioning.
To pull in the latest version, you can use either command-line [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) or [yarn](https://classic.yarnpkg.com/lang/en/docs/install/).

:::note

There are several peer-dependencies that you may already have in your project, such as ethers, react, and react-dom.
The following examples show how to pull in the `ethers` peer dependency.

:::

```shell
$ npm install @rarimo/rarimo-settlement-component ethers
```

```shell
$ yarn add @rarimo/rarimo-settlement-component ethers
```
